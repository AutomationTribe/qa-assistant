# UI Design Files

This folder contains HTML mockup files for every page in Regi.
Open them in any browser — they are standalone design references.
They are not imported into the app and contain no real functionality.

## How to use
1. Open any .html file in your browser to review the design
2. If approved, create an issue file in context/issues/ to implement it
3. Reference this folder when running context/ui-design-generator.md

## Files

| File | Pages covered | Status |
|------|--------------|--------|
| regi-ui-design.html | Login, Register, Projects, Add Project | ✅ Approved |
| template-builder-design.html | Template Builder, Add Field Modal, Preview | ✅ Generated |

## Design system
Primary colour:   #4F46E5
